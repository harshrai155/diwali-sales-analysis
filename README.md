# 🪔 Diwali Sales Analysis

## 📌 Project Overview
This project analyzes Diwali sales data to understand customer behavior, purchasing patterns, and key business insights. The analysis helps businesses identify target customers and improve sales strategies during festive seasons.

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Key Analysis Performed
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Customer segmentation based on:
  - Gender
  - Age Group
  - Marital Status
  - State
  - Occupation
- Product category analysis
- Sales trend visualization

---

## 📈 Key Insights
- Married women aged 26–35 are the most frequent buyers
- Customers from Uttar Pradesh, Maharashtra, and Karnataka contribute the highest sales
- IT, Healthcare, and Aviation professionals have higher purchasing power
- Food, Clothing, and Electronics are the most popular product categories

---

## 📂 Project Files
- Diwali_Sales_Analysis.ipynb → Main analysis notebook
- Diwali Sales Data.csv → Dataset used

---

## 🚀 Conclusion
The analysis shows that targeting married women aged 26–35 from specific states and professions can significantly improve sales during festive seasons like Diwali.
